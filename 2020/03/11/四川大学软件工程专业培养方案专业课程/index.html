四川大学软件工程专业所学专业课程

一、学科基础课程：

  1. 线性代数

  2. 微积分

  3. 大学物理

  4. 数学逻辑：应用与设计

  5. 离散数学

二、专业核心课程（必修）：

  1. 程序设计基础
  2. 计算机系统导论
  3. 面向对象程序设计导论
  4. 数据结构与算法
  5. 计算机组成和体系结构
  6. 数据库系统和信息管理
  7. 软件工程导论
  8. 计算机网络和分布式系统

三、专业核心课程（选修）：

（1）软件工程通用方向

  1. 软件开发环境与工具
  2. 软件工程社会和职业问题
  3. 软件需求分析
  4. 软件设计与体系结构
  5. 软件构造
  6. 软件质量保证和测试
  7. 团队激励与沟通
  8. 软件工程经济学

（2）移动/Web应用开发方向

  1. .NET和.NET Core
  2. 分布式系统与互联网
  3. 移动编程
  4. Web应用开发
  5. 无线网络技术
  6. 人机交互概论
  7. 服务器虚拟化技术

（3）图形图像处理方向

  1. 计算机图形学
  2. 图论
  3. 数字图像处理
  4. 医学成像
  5. 多媒体技术
  6. 深度学习
  7. 模式识别

（4）信息系统与数据分析方向

  1. 机器学习
  2. 深度学习
  3. 大数据技术原理与应用
  4. 信息系统分析与设计
  5. 商务数据分析
  6. 运筹学
  7. 数据挖掘技术概论

（5）软件工程研究方向

  1. 软件工程新技术专题
  2. 专业写作
  3. 软件工程研究方法
  4. 软件工程形式化方法
  5. 软件工程讨论会
  6. 并行建模和分析
  7. 文献阅读

（6）其它

  1. 嵌入式系统
  2. 编译原理
  3. 中间件技术
  4. 人工智能导论
  5. AI面向语言编程
  6. 并行处理
  7. 算法分析与设计

四、实践教育课程：

  1. 软件项目管理
  2. 程序设计实践1
  3. 程序设计实践2
  4. IT企业实训
